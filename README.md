# Diet & Health

Strona internetowa dla dietetyka/specjalisty zdrowia. Zbudowana w Next.js App Router.

## Stack

- [Next.js 15](https://nextjs.org/) — App Router, Server Components
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Radix UI](https://www.radix-ui.com/) — komponenty UI
- [Supabase](https://supabase.com/) — baza danych
- [next-auth](https://next-auth.js.org/) — autoryzacja
- [@tanstack/react-query](https://tanstack.com/query) — cache i fetching danych
- [Vercel Analytics](https://vercel.com/analytics)

## Skrypty

- `dev` — serwer deweloperski
- `build` — build produkcyjny
- `typecheck` — sprawdzanie typów TypeScript
- `lint` — ESLint
- `prettier:check` — sprawdzanie formatowania
- `prettier:write` — formatowanie plików
